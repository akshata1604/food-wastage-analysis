# Food Wastage Analysis – SQL & Streamlit Application

## Overview
This project analyzes food wastage patterns using SQL-based data analysis and presents insights through an interactive Streamlit dashboard.

The objective is to identify inefficiencies, wastage trends, and operational insights to support better decision-making.

---

## Problem Statement
Food wastage impacts operational cost and sustainability. This project analyzes structured data to:
- Identify high-wastage categories
- Track wastage trends
- Analyze regional and operational inefficiencies
- Support data-driven decision making

---

## Project Workflow

1. Data Cleaning & Preparation
2. SQL-Based Analysis
   - Joins
   - Aggregations (GROUP BY, HAVING)
   - Filtering & Conditional Queries
   - Business KPI Extraction
3. Insight Generation
4. Interactive Dashboard using Streamlit

---

## Key Features

- SQL-based business analysis
- Aggregated KPI calculations
- Interactive dashboard
- Real-time filtering & visualization
- Structured data exploration

## How to Run Locally

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the Streamlit app:
   streamlit run app.py
